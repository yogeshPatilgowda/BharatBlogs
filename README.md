# 🇮🇳 BharatBlogs

A modern blogging platform where users can create, manage, and explore blogs through a clean and responsive interface.

## ✨ Features
- User Authentication
- Create, Edit & Delete Blogs
- View All Blogs
- Responsive UI
- Secure Backend

## 🛠 Tech Stack
- React.js
- Node.js
- Express.js
- MongoDB
- JWT Authentication

## 🚀 Getting Started

```bash
git clone https://github.com/yogeshPatilgowda/BharatBlogs.git
cd BharatBlogs
npm install
npm start
```
